## Descripcion: 
Una agencia de vehículos necesita un sistema sencillo para registrar y consultar vehículos disponibles.

El sistema deberá permitir almacenar información básica de diferentes vehículos y mostrarla mediante un menú interactivo.

---

# Menú del Sistema

```text
====================================
      REGISTRO DE VEHÍCULOS
====================================

1. Registrar vehículo
2. Mostrar vehículos
3. Buscar vehículo por placa
4. Mostrar vehículos por marca
5. Salir
```

---

# Requisitos Funcionales

## 1. Registrar vehículo

El sistema debe permitir registrar:

- Placa
- Marca
- Modelo
- Año
- Color
- Precio

---

## 2. Mostrar vehículos

Mostrar todos los vehículos registrados.

---

## 3. Buscar vehículo por placa

Permitir buscar un vehículo específico utilizando su número de placa.

---

## 4. Mostrar vehículos por marca

Permitir mostrar todos los vehículos pertenecientes a una marca específica.

---

# Requisitos Técnicos

El proyecto debe implementar:

- Clase Vehiculo
- Constructores
- Métodos
- Encapsulamiento
- Arrays o ArrayList
- Menú interactivo
- Validaciones básicas

---

# Restricciones

- No se permite realizar toda la lógica dentro del método main.
- El sistema debe estar dividido en clases.
- Debe utilizar programación orientada a objetos correctamente.
